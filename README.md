# Full-Stack App

This workspace contains a Python backend and a React + Vite frontend.

## Backend

- Located in `backend/` folder
- Built with FastAPI
- Run with: `cd backend && .\venv\Scripts\activate && uvicorn main:app --reload`
- API docs available at http://localhost:8000/docs

## Frontend

- Located in `frontend/` folder
- Built with React and Vite
- Run with: `cd frontend && npm run dev`
- Opens at http://localhost:5173

## Getting Started

1. Start the backend: Follow the backend instructions above.
2. Start the frontend: Follow the frontend instructions above.
3. The frontend can now call the backend API.
